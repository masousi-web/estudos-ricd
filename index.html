Como usar (bem simples):
1) Salve este arquivo como index.html na mesma pasta dos resumos.
2) Para cada título, crie (ou adicione depois) os arquivos no formato: titulo-i.html e titulo-i-kindle.html (e assim por diante).
3) Para criar um “Outro módulo”, duplique um card e troque o href para o nome do seu novo HTML.
